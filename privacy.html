Privacy Policy

This application accesses Pinterest data only with user authorization via OAuth.

Data Collected

  • Board names and descriptions
  • Saved pin metadata (e.g., descriptions, image URLs)

Purpose

The data is used solely to generate personalized Spotify music recommendations.

Data Storage

User data is not permanently stored. Data is processed temporarily for recommendation purposes.

Third Parties

This application integrates with the Pinterest and Spotify APIs. It is not affiliated with Pinterest or Spotify.
