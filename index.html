Pinterest → Spotify Music Recommender

This application analyzes a user’s Pinterest boards and saved pins to generate personalized Spotify music recommendations.

Using semantic embeddings from pin metadata (board names, descriptions, and images), we map visual interests to Spotify audio features such as:

  • Acousticness
  • Energy
  • Danceability
  • Valence
  • Instrumentalness

This is a student research project and is not affiliated with Pinterest or Spotify.
