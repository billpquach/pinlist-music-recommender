Pinlist

A multimodal music recommendation system based on Pinterest user interests.

Pinlist is a student research project that analyzes a user’s Pinterest boards and saved pins to generate personalized music recommendations using Spotify audio features.

How It Works

The system extracts semantic information from board names, descriptions, and available pin metadata. These signals are encoded into embeddings and mapped to Spotify audio characteristics through a predictive model.

Predicted Audio Features

  • Acousticness
  • Energy
  • Danceability
  • Valence
  • Instrumentalness

The application integrates with the official Pinterest and Spotify APIs using user authorization. Data is processed only for recommendation generation.
